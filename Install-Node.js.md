### Goal

Install Node.js 0.10 (or better) in order to run the `sagan-client` build.

### Prerequisites

_None_

### Steps

1. Go to [nodejs.org](http://nodejs.org) and click "Install"
2. Verify that the installation was successful:
```
$ node --version
v0.10.26
```
